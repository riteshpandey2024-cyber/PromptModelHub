# PromptModelHub

PromptModelHub is a curated collection of **system prompts, agent prompts, and configurations used by modern AI tools and coding assistants**.

This repository is designed for **developers, researchers, and prompt engineers** who want to explore how different AI systems are instructed and structured.

By studying these prompts, users can understand **prompt engineering techniques, AI assistant behavior design, and agent architecture** used across multiple platforms.
